Yes — always. I'll do a free 15-minute call to understand your problem and give you an honest estimate before any commitment.
What if you can't fix it?
If I can't fix your problem, you don't pay. I'll always be upfront about what's possible before I start, so there are no surprises.
Do you work on both Macs and PCs?
Yes — I work on Windows and macOS computers, as well as iPhones, Android phones, and tablets. If it has a screen and a power button, I can likely help.
Will you install something I already bought?
Absolutely. Printers, routers, smart TVs, external drives — if you bought it and need it set up, I'll come install and configure it for you.
Do you work on weekends?
Yes. I offer flexible scheduling including evenings and weekends — because tech problems don't keep business hours.
